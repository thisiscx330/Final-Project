影像處理概論 期末報告

實驗結果

I.	CCL_OCR.py

框出車牌中的文字，用OCR辨識車牌號碼
 
辨識成功   (這張車牌的英文字I會被辨識成1，但台灣的車牌不會有英文I, O，所以應該可以避免這個錯誤吧……)

II.	LCR.py

強化原始圖片，再OCR辨識
 
辨識成功

 	使用OCR辨識車牌號碼需要很乾淨的圖片，才能辨識的很準確。
  
 	要正確的擷取到車牌號碼和訓練的模型有很大的影響。車牌大小、歪斜、不清晰、光線落差、反光等很多問題都會影響辨識的結果。
  
 	如果停車場有適當的光線照明，照片拍攝角度跟大小也能固定，OCR應該能辨識得比較好。
